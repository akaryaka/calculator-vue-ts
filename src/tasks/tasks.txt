узнал 
- добавление сниппета для создания базового шаблона
  Самый распространенный способ — настроить User Snippets в VS Code.
  Ctrl+Shift+P -> Snippets -> vue.json/vue
  Добавьте шаблон для <script setup> (современный подход): 
  json
  "Vue 3 Setup": {
    "prefix": "v3s",
    "body": [
      "<template>",
      "  <div class=\"$1\">",
      "    $0",
      "  </div>",
      "</template>",
      "",
      "<script setup lang=\"ts\">",
      "",
      "</script>",
      "",
      "<style scoped>",
      "",
      "</style>"
    ],
    "description": "Vue 3 Component with script setup"
  }

  Теперь в новом .vue файле, набрав v3s и нажав Tab, вы получите полный шаблон.
